(2) What is the difference between State and Props?
answer :- Props are read-only data passed from parent to child. State is mutable data managed inside a component.

(3)What is the useState hook, and how does it work?
answer :- useState is a React Hook that lets you add and update state in a functional component using a state value and a setter function.

(4) How can you share state between components in React?
answer :- You can share state in React by lifting state up to a common parent and passing it down via props (or using Context API for global state).

(5) How is event handling done in React?
answer :- Event handling in React is done by passing functions as event handlers (like onClick) using camelCase syntax, and React handles events through its Synthetic Event system.
